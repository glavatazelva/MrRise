# MrRise
My Android application. A twist on a basic alarm clock where the user needs to beat one of the 3 minigames to shut down the alarm clock.
## Features
- 3 unique puzzles (Simon Says, Math Challenge, QR Code Scanner)
- Multiple alarm sounds (Default, Vibration, Phone Ringtone)
- Repeating and one-time alarms
- Swipe to delete
- Edit existing alarms
- Multi-language support (English, Croatian, German)
- Dark/Light theme support

## Tech Stack
- Kotlin
- MVVM Architecture
- Room Database
- View Binding
- Coroutines

## How to Build
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or device

## Requirements
- Android Studio
- Minimum SDK: API 24 (Android 7.0)
- Target SDK: API 34
